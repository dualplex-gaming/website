What is Dualplex?

Dualplex is an ARM-based dualscreen gaming handheld system.


Console-like convenience with PC-level flexibility

Dualplex not only features an easy & convenient-to-use, full console software experience, but it also has the flexibility and freedom of a full desktop computer!


Powered by cafeOS, brewed for both tinkerers & casuals alike!

cafeOS is an open-source, Android-based operating system specially designed for Dualplex, and includes cafeDX (cafe Dual eXperience), the Dualplex frontend/UI app.

You can check out the Dualplex GitHub organization by clicking this text!


Dual screens. Absolute peak gaming.

Dualplex's hardware features a 21:9 aspect ratio top display, and a 16:9 bottom display. Both displays are touch screens and support original, exclusive games.


!! Dualplex is in an early conception phase. It is currently not an existing real product yet, but currently private development on cafeDX & cafeOS is active. More details coming soon!
